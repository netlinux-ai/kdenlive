kdenlive

Kdenlive 25.12 — free and open source video editor based on MLT Framework and KDE Frameworks, with bundled MLT 7.34.1.

Why this build?

Ubuntu 24.04 ships Kdenlive 23.08.3, a KDE Frameworks 5 / Qt5 build that is now two major release series behind upstream. This build provides Kdenlive 25.12.2 (February 2026), a substantial upgrade:

  • KF5/Qt5 → KF6/Qt6 — modern toolkit with improved Wayland support, better HiDPI scaling, and native platform integration
  • New docking system — KDDockWidgets replaces the legacy panel system, making workspace layout rearrangement far easier
  • Enhanced audio display — waveform overview in the clip monitor for precise audio editing
  • Improved effects engine — faster rendering, new effects, and better keyframe handling
  • Nested timelines — sequence clips allow embedding timelines within timelines for complex project organisation
  • Speech-to-text — integrated subtitle generation from audio tracks
  • MLT 7.34.1 — bundled multimedia framework with OpenCV integration, vidstab stabilisation, and frei0r effects (Ubuntu 24.04 ships MLT 7.22 which lacks many of these features)
  • Two years of bug fixes — hundreds of stability improvements, crash fixes, and workflow enhancements since 23.08
